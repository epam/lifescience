####################
Timeout options
####################

.. indigo_option::
    :name: timeout
    :type: integer
    :default: 0
    :short: Timeout (is ms) for Indigo operation. Zero means no timeout.

.. indigo_option::
    :name: aam-timeout
    :type: integer
    :default: 0
    :short: Timeout (is ms) for atom-to-atom mapping computation operation. Zero means no timeout.
