# Indigo 1.39.0
Released 2026-02-20

## Features
* [3211](https://github.com/epam/Indigo/issues/3211) Introducing alignment of sense and antisense chains
* [3273](https://github.com/epam/Indigo/issues/3273) Circular structures improvement - formation of fused rings 
* [3250](https://github.com/epam/Indigo/issues/3250) Alignment of sense and antisense strings in AxoLabs format
* [3337](https://github.com/epam/Indigo/issues/3337) Bingo NoSQL tautomer substructure search support.
